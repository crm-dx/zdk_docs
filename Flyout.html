Flyout

Class representing a Flyout Object.

Functions

#

open(config, data)

Open Widget component inside a Flyout

Parameters:
Name Type Attributes Description
config object

Flyout type config
type String

flyout type (widget)
id String

id of the widget (Custom Button type)
data Any <optional>

data to be passed into the widget
Example

SampleCopycopied

ZDK.Client.getFlyout('myFlyout').open({ id: '111111000000145034', type: 'widget' }, { data: 'sample data' });

#

notify(data, options)

Notify flyout

Parameters:
Name Type Attributes Description
data Any

data to be passed to the flyout
options Object <optional>
wait String <optional>

true/false - wait for response from flyout
Example

SampleCopycopied

var response = ZDK.Client.getFlyout('myFlyout').notify({data: 'sample data'}, {wait: true});
